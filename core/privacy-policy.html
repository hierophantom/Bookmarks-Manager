Privacy Policy

Last updated: March 12, 2026

Bookmark Manager is designed to help you organize and access your bookmarks. This policy explains what data the extension accesses and how it is used.

Data We Access

  • Bookmarks data: to read, create, update, move, and delete bookmarks.
  • Tabs and tab groups: to open bookmarks and organize opened tabs in groups.
  • Optional browsing history: only if you use features that suggest or import from history.
  • Website content and user interactions: only to support user-invoked search and overlay features on pages.
  • Local settings: preferences such as theme, layout, and search engine are stored locally.

How We Use Data

  • To provide bookmark management features requested by the user.
  • To render and personalize extension UI behavior.
  • To improve convenience features such as search overlays and tab workflows.

Data Sharing

We do not sell user data. Data is not shared with third parties except where needed for a user-requested feature, such as loading background images from Unsplash or fetching quote content.

Data Storage and Retention

Preference data is stored in browser extension storage. You can clear extension data at any time by removing the extension or resetting its settings.

Security

We use the permissions required for core extension functionality and aim to minimize access whenever possible.
