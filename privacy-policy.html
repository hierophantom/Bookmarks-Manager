Privacy Policy

Last updated: March 12, 2026

Bookmark Manager helps users organize and access browser bookmarks. This policy explains what data the extension accesses and how it is used.

Data We Access

  • Bookmarks data: to read, create, update, move, and delete bookmarks.
  • Tabs and tab groups: to open bookmarks and organize opened tabs in groups.
  • Optional browsing history: only if users use features that suggest or import from history.
  • Website content and user interactions: only to support user-invoked search and overlay features.
  • Local settings: preferences such as theme, layout, and search engine are stored locally.

How We Use Data

  • To provide bookmark management features requested by the user.
  • To render and personalize extension UI behavior.
  • To support convenience features such as search overlays and tab workflows.

Data Sharing

We do not sell user data. Data is not shared with third parties except where needed for user-requested features, such as loading background images from Unsplash or fetching quote content.

Data Storage and Retention

Preference data is stored in browser extension storage. Users can clear extension data at any time by removing the extension or resetting settings.

Security

We request permissions required for core functionality and aim to minimize access whenever possible.
